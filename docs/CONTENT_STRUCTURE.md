# 📚 Structure des Contenus Pédagogiques - BREVET MASTER

Ce document décrit comment le programme officiel du Brevet (DNB) est intégré et structuré dans la plateforme.

## 🗂️ Organisation par Matières
Le MVP (Phase 1) se concentre sur les matières à plus fort coefficient :
1. **Mathématiques** (Coeff 3)
2. **Français** (Coeff 3)
3. **Histoire-Géographie / EMC** (Coeff 3)
4. **Sciences (SVT / Physique)** (Coeff 2)

## 🏗️ Structure d'une Notion
Chaque notion (ex: *Théorème de Thalès*) est organisée selon le flux suivant :
1. **Onboarding** : Présentation des objectifs et prérequis.
2. **Leçon interactive** : Micro-contenu (slides ou texte court) avec questions de vérification rapide.
3. **Quiz de Validation** : Test final permettant de gagner de l'XP et des Gemmes.
4. **Analyse ARIA (V2)** : Feedback personnalisé sur les erreurs commises via RAG.

## 📝 Types de Questions Implémentés
- **QCM Classique** (Implémenté en V1) : Sélection de la bonne réponse parmi 4 options.
- **Vrai / Faux** : Validation d'une affirmation.
- **Réduction/Simplification** (Prévu) : Questions ouvertes courtes évaluées par IA.

## 🎯 Alignement Pédagogique (Taxonomie de Bloom)
Chaque activité est tagguée par niveau de difficulté :
- **N1 - Mémorisation** : Rappel de formules (ex: def de Thalès).
- **N2 - Compréhension** : Reformuler une règle.
- **N3 - Application** : Résoudre un exercice type (ex: calcul de longueur).
- **N4 - Analyse** : Cas complexe (ex: droites non parallèles).
